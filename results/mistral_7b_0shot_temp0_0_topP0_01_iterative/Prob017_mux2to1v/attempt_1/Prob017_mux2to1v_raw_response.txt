 Here is the Verilog code for a 2-1 multiplexer as per your description:

```verilog
module TopModule (
  input wire [99:0] a,
  input wire [99:0] b,
  input wire sel,
  output reg [99:0] out
);

always @(posedge clk) begin
  if (sel)
    out <= b;
  else
    out <= a;
end
```
