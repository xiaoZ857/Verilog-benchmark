 Here is the Verilog code for the TopModule as per your description:

```verilog
module TopModule (
  input wire clk,
  input wire areset,
  input [7:0] d,
  output reg [7:0] q
);

// Instantiate 8 D flip-flops with active high asynchronous reset
always @(posedge clk or negedge areset) begin
  if (areset) begin
    q <= 8'b0; // Reset all DFFs to 0 when areset is high
  end else begin
    q <= d; // Assign the value of 'd' to 'q' on positive edge of clk
  end
end

endmodule
```

This code creates an 8-bit register (D flip-flop) array with active high asynchronous reset. The `areset` signal resets all DFFs to 0 when it is high, and the values from the `d` input are assigned to the `q` output on the positive edge of the `clk` signal.
