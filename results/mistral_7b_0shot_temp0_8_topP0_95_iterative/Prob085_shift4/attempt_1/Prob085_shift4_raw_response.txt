module TopModule (

  input areset,

  input [3:0] data,
  output reg [3:0] q

always @(posedge clk) begin

    q <= 4'b0; // Reset the shift register to zero

      q[0] <= 0; // The first bit is reset
    end
  end
end

  if (areset) q <= 4'b0; // Synchronous reset for q output

initial begin
  q = 4'b0; // Initialize the shift register to zero
end

This Verilog code creates a 4-bit shift register with asynchronous positive edge triggered `areset`, synchronous active high signals `load` and `ena`, and an output `q`. The shift register will be reset to zero when `areset` is asserted, and it will load data from the input `data` when both `load` and `ena` are asserted. If only `ena` is asserted, the shift register will right-shift its contents on each positive clock edge. The output `q` has a synchronous reset for added safety.
